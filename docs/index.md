# Getting Started

**JSON data parser for Zig**

Jsonic is intended to be a high-level abstraction for JSON data parsing. Jsonic is just a wrapper on top of Zigs `std.json` module.

If you need more functionality or code coverage, Please create an issue at [**Jsonic Repo**](https://github.com/bitlaab-bolt/jsonic) and buy us some coffee.
