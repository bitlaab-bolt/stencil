# jsonic

**JSON data parser**

Jsonic is intended to be a high-level abstraction for JSON data parsing. Jsonic is just a wrapper on top of Zigs `std.json` module.

## Platform Support

Jsonic has cross-platform support because of `std.json` module.

## Dependency

Jsonic has no external dependencies.
